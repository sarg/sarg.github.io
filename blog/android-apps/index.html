Android apps that I use

2020/04/24

My phone is running AOSP without google services and almost all of the applications I use come from F-Droid. Few exceptions are apps for paid services that I use and they are installed from Aurora Store.

  • I use fdroidcl to install and update applications.
  • Backup is covered in post “Backing up Android phone".
  • I use syncthing to share a directory between my laptops and phone.
  • I have “Password Store” files in the shared directory in case I need access to passwords on phone (rarely).
  • In AFWall+ I whitelist only selected apps. All other are denied to access net.
